# 对象存储 S3/OSS

## 什么是对象存储

对象存储是专门存放文件的云服务。图片、视频、文档、备份文件都适合放在对象存储里。

和服务器硬盘的区别：对象存储容量无限、自动备份、通过 HTTP 访问、按量计费。

## 核心概念

| 概念 | 说明 |
|------|------|
| Bucket | 存储桶，类似文件夹 |
| Object | 对象，就是文件 |
| Key | 对象的路径/名称 |
| ACL | 访问控制 |
| 预签名 URL | 临时授权的访问链接 |

## 常见用途

- 用户上传的图片和文件
- 静态网站托管
- 日志存储
- 数据库备份
- 大文件分发（配合 CDN）

## 上传流程（安全方式）

```
1. 前端请求后端获取预签名 URL
2. 后端生成预签名 URL 返回
3. 前端直接上传文件到对象存储
4. 上传完成后通知后端保存文件信息
```

这样文件不经过你的服务器，节省带宽和服务器资源。

## 主流服务

| 服务 | 提供商 |
|------|--------|
| S3 | AWS |
| OSS | 阿里云 |
| COS | 腾讯云 |
| R2 | Cloudflare（免流量费） |
| MinIO | 自建（S3 兼容） |

> 对象存储是现代应用的标配。文件不要存在服务器本地，放到对象存储里。
