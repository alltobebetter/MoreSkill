# 机器学习是什么

## 核心思想

传统编程：你告诉计算机规则，它按规则执行。
机器学习：你给计算机数据，它自己学出规则。

比喻：教小孩认猫。你不会给他一个"猫的定义"，而是给他看很多猫的照片，他自己就学会了。

## 三种学习方式

### 监督学习

给带标签的数据，让模型学习输入和输出的关系。

```
输入：房屋面积、楼层、位置
输出：房价
→ 模型学会：面积大 + 好位置 = 高房价
```

应用：分类（垃圾邮件检测）、回归（价格预测）

### 无监督学习

给没有标签的数据，让模型发现数据中的模式。

```
输入：用户购买记录
输出：用户分群（高消费群、低频群...）
```

应用：聚类、降维、异常检测

### 强化学习

通过试错和奖励来学习最优策略。

```
智能体在环境中行动 → 获得奖励或惩罚 → 调整策略
```

应用：游戏 AI、机器人控制、推荐系统

## 常见算法

| 算法 | 类型 | 适用场景 |
|------|------|---------|
| 线性回归 | 回归 | 价格预测 |
| 逻辑回归 | 分类 | 二分类问题 |
| 决策树 | 分类/回归 | 可解释性要求高 |
| 随机森林 | 分类/回归 | 通用，效果好 |
| K-Means | 聚类 | 用户分群 |
| KNN | 分类 | 推荐系统 |

## 机器学习流程

```
数据收集 → 数据清洗 → 特征工程 → 模型训练 → 模型评估 → 部署上线
```

其中数据清洗和特征工程占了 80% 的工作量。

## 常用工具

| 工具 | 用途 |
|------|------|
| scikit-learn | 传统 ML 算法 |
| pandas | 数据处理 |
| numpy | 数值计算 |
| matplotlib | 数据可视化 |
| Jupyter Notebook | 交互式开发 |

> 机器学习不是魔法，它的本质是"从数据中找规律"。数据质量决定了模型的上限。
